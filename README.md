# Fuel Economy ⛽

## Overview
This project was given to me in my MAT-303 course which is Applied Statistics III. I was asked to analyze a data set that could be used to study the fuel economy of a car, which I will be using to create models to predict fuel economy. This project showcases skills on creating data visualizations (histograms, scatterplots, Q-Q plots) and gathering statistics like correlation, summary statistics, and confidence intervals.

## Objective
The objective is to create a multiple regression model for the mtcars data set, using miles per gallon as the dependent variable, and rear axle ratio and horsepower as the independent variables.
